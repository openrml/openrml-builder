# RML Examples

This directory contains `.rml.txt` role definitions exported from the RolesAI constructor.

These files demonstrate valid RML documents in practice. They can be:
- Imported directly into the constructor UI
- Pasted into any LLM chat to activate the role
- Used as reference for building a compliant parser
- Tested against a compliance validator

---

## How to Add Examples

1. Open the role in the constructor
2. Set `Status: published`
3. Click **Export RML** â€” saves `role-name.rml.txt`
4. Add the file to this directory following the naming convention below
5. Update the table in this README

**Naming convention:** `role-name-kebab-case.rml.txt`
